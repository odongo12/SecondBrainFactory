Sign in

There's no server here to check a password against — GitHub Pages only serves static files. The only thing that can actually authorize a saved change is a GitHub access token with write permission to this repo, so that's what "login" means on this page. Use a fine-grained personal access token scoped to only this repository, with Contents: Read and write permission — not a classic all-repos token. The token stays in your browser and is sent only to api.github.com, never anywhere else.
